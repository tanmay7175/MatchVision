How to Run

Clone the Repository

git clone https://github.com/tanmay7175/MatchVision.git
cd MatchVision

python -m venv venv
venv\Scripts\activate   # For Windows

pip install -r requirements.txt

python main.py

streamlit run streamlit_app.py
